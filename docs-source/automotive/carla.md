# CARLA Simulation

## Overview

Experiments with CARLA focused on vehicle behaviour and ADAS concepts, including vehicle spawning and control, speed ramps, adaptive-cruise ideas, lane detection, lane keeping, maps, and bird’s-eye visualisation.
