# 8-bit CPU

## Overview

A small CPU built in Logisim to understand how instructions move through registers, an ALU, RAM, flags, and control logic.

## Implemented concepts

- Accumulator/register data path
- Arithmetic and logic operations
- RAM reads and writes
- Instruction sequencing
- Zero flag
- Clocked state changes

## Next steps

- Conditional jumps using the zero flag
- A halt instruction
- A clearer instruction-set reference
- Example programs and execution traces
