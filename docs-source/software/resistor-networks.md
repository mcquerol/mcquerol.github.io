# Resistor Network Modelling

## Overview

A C++ project for representing and analysing resistor networks programmatically. It connects object-oriented software design with electrical circuit analysis.

## Documentation plan

- Network representation
- Component and node model
- Series and parallel behaviour
- Calculation approach
- Example networks
- Validation and limitations
